# 架构决策 - CLIP图像搜索重构方案

**决策日期**：2025年12月10日

## 背景
当前CLIP图像搜索实现存在根本性错误，使用文本匹配而非特征向量搜索，导致返回提示词本身而非真实的图像理解结果。需要重构为基于CLIP特征向量的图像搜索架构。

## 解决方案
采用完全基于特征向量的CLIP搜索方案。索引构建时为所有图片提取CLIP特征向量存入Faiss索引，图片搜索时提取上传图片的特征向量，通过Faiss向量搜索找到最相似的图片。删除错误的文本匹配逻辑，确保图像搜索基于真正的语义特征向量比对。

## 影响
需要重建所有图片索引以包含CLIP特征向量，索引构建时间增加但搜索效果显著提升。搜索性能基于向量计算速度较快，准确性大幅提高。依赖Faiss向量搜索能力的稳定性。
